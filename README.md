<h3>Доступные скрипты:</h3>
<ul>
    <li>
        Для запуска DevServer: <br>
        <code>npm start</code>
    </li>
    <li>
        Для сборки в production моде: <br>
        <code>npm run build</code>
    </li>
    <li>
        Для сборки в development моде: <br>
        <code>npm run dev</code>
    </li>
</ul>
